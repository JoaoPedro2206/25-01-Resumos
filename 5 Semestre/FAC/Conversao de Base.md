#### Decimal para Binario
- Divide o numero por 2 ate dar 0;
- O numero em binario vai ser os resto do ultimo para o primeiro;
Ex: 54 para binario
54/2 = 27 resto = 0
27/2 = 13 resto = 1
13/2 = 6 resto = 1
6/2 = 3 resto = 0
3/2 = 1 resto = 1
1/2 = 0 resto = 1
